jQueryのメソッド

テキストの変更と取得

テキストの変更
$('h1','p'や'id名','クラス名'など).text('changed!');
( )内に書きたい文字列を入力する
<h1>変更前</h1>
<h1>changed!</h1>
テキストの取得
$('h1','p'や'id名','クラス名'など).text( );
( )内を空欄にして取得する
また取得した文字列をそのまま入れることもできる
$('p').text($('h1').text( ) );
<p></p>

HTMLの変更と取得

HTMLの変更
$('h1','p'や'id名','クラス名'など).html(<strong>changed!</strong>);
( )内に変更したいHTMLを入力する
変更前
こんにちは
$('#change-html').html('<strong>こんばんわ！</strong>');
変更後
HTMLの取得
$('h1','p'や'id名','クラス名'など).html( );
( )内を空欄にして取得する
また取得した文字列をそのまま入れることもできる
変更前

$('#get-html').html($('#change-html').html( ) );

属性値

属性値の変更
属性値を変更できる
$('セレクター').attr('属性名','属性値');
変更前
googleへ
変更後
$('a').attr('href','https://www.yahoo.co.jp/');
googleへ(実際のリンクはgoogleではなくyahoo!になっている)
属性値の取得
$('セレクター).attr('属性名');
( )内に属性名を指定して取得する
また取得した文字列をそのまま入れることもできる
$('a').text($('#google').attr( 'href') );

class属性

class属性の追加
addClass( )を使う事でclassを追加できる
$('セレクター').addClass('class名');
class属性の削除
removeClass( )を使う事でclassを削除できる
$('セレクター').removeClass('class名');

CSSプロパティ

CSSプロパティの設定
css("プロパティ名","値")を使うとプロパティの値を変えたり取得できたりする
変更前

変更後
$('#change-css').css("color","red");

<h3>テキストテキストテキストテキストテキスト</h3>

複数のプロパティを変える場合
$('').css({"プロパティ名" : "値" ,

※プロパティ名と値の間を : ( コロン )で区切る事と , ( カンマ )で繋げる事に気を付ける
$('#change-css').css({"color":"red",
           "backgroundColor":"yellow",
           "display":"inline"});

CSSプロパティの取得

要素の選択

parent( )/children( )
parent( )は指定した親要素、children( )は指定した子要素
next( )/prev( )
next( )は指定した次の要素を、prev( )指定した前の要素を指定する
$('#main>li:eq(1)').next( ).css('color','green');
id="main"直下の2番目(eq1)のlistの後ろ
$('#main>li:eq(2)').prev( ).css('color','skyblue')
id="main"直下の3番目(eq2)のlistの手前
siblings( )
siblings( )は指定した物の兄弟要素を指定する
$('#sub>li:eq(1)').siblings( ).css('color','orange');
id="sub"直下のlistの2番目(eq1)の兄弟要素
